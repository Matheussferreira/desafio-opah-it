# Ansible

## Visão geral
Este diretório contém um playbook de hardening para VMs Linux on-prem que suportam integração com a nuvem.

## O que o playbook faz
- Atualiza pacotes do sistema.
- Habilita firewall básico.
- Instala e configura `fail2ban`.
- Instala e configura `auditd`.
- Configura atualizações automáticas.
