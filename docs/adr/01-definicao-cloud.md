# Escolha do GCP como cloud principal

## Contexto
A empresa está em transformação digital e busca uma nuvem que ofereça serviços gerenciados fortes para integração com on-prem e governança centralizada.

## Decisão
Adotar o Google Cloud Platform como cloud primária para este desafio.

## Próximos passos
- Uso de serviços gerenciados como Pub/Sub, Cloud Run e Cloud SQL.
- Redução de esforço operacional em infraestrutura de middleware.
- Dependência de recursos e políticas do GCP.

## Alternativas consideradas
- AWS: Considerei, mas aumentaria a complexidade de integração com o ambiente e falando de FinOps, aumentaria custo.
- Azure: Menos aderente ao modelo de serviços orientados a eventos proposto.
